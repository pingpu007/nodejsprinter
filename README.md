## ESCPOS PROJECT
ESC/POS Printer driver for Node.js (Thai printing)

## Installation

### Dependency
```bash
"node": v14.15.4
"npm": v6.14.10
"escpos": ^3.0.0-alpha.6,
"escpos-usb": ^3.0.0-alpha.4
"express": ^4.17.1
"iconv": ^3.0.0
```
### git
```bash
$ git clone https://github.com/pingpu007/nodeescpos
```

### Linux/Ubuntu
if escpos-usb 
```bash
sudo apt-get install build-essential libudev-dev
```
```bash
$ npm install
```
